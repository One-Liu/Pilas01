# Pilas01
Ejercicio donde se hace uso de una pila.

1. Inicializar pila
2. Ingresar dato
3. Eliminar dato
4. Mostrar cima
5. Mostrar datos de la pila
6. Total numeros pares
7. Total numeros impares
8. Suma de todos los valores de la pila
9. Ordenar los valores de la pila (ascendente)
10. Vaciar pila
11. Salir
